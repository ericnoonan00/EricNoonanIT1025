HTML Links

HTML Links are used to send the user to a specific website. They come in handy for websites with many pages in that you are able to send the user to a different page with them. Also, links can be used to download something. A link is programmed using the <a> tag with the href attribute. Specific targets can also be attributed to the link such as a blank tab using the target attribute.
